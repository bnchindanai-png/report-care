# ระบบรายงานการปฏิบัติหน้าที่ดูแลนักเรียน (React Version)

## การติดตั้ง

1. ติดตั้ง Node.js จาก https://nodejs.org/
2. ติดตั้ง dependencies:
   ```bash
   npm install
   ```

## การรันโปรเจค

### Development Mode
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

### Preview Production
```bash
npm run preview
```

## ฟีเจอร์หลัก

- ✅ รายงานประจำวัน
- ✅ รายงานทั้งหมด  
- ✅ จัดการข้อมูล (แก้ไข/ลบ)
- ✅ การรับทราบของผู้อำนวยการ
- ✅ ระบบความปลอดภัย (รหัสผ่านแยกกัน)
- ✅ แสดงสถานะการรับทราบ
- ✅ รองรับการอัปโหลดรูปภาพ
